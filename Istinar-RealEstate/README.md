# Istinar Real Estate

منصة SaaS عقارية مستقلة تحول المخططات والفرص إلى تجربة تفاعلية قابلة للإدارة والقياس واتخاذ القرار.

## Structure

- `backend/` — Django REST API, authentication, business rules, and admin.
- `frontend/` — Next.js customer, provider, entrepreneur, and platform-admin experiences.
- `docs/` — product and technical delivery documentation.
- `infra/` — deployment configuration and environment examples.
